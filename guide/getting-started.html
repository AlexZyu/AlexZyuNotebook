# 静态资源

可以把一些静态资源放在.vuepress/public目录中，它们会被复制到最终生成的网站的根目录下。

在下列这些情况中，可能会用到它：

  • 可能需要提供一些静态资源，但是它们并不直接被某个 Markdown 文件引用，比如 favicon 和 PWA 图标。
  • 可能想要托管一些共享的静态资源，甚至可能需要在你的网站外部引用它，比如 Logo 图片。
  • 可能想在你的 Markdown 内容中通过绝对路径来引入图片。

# 发布到Github Pages

当前发布在 https://alexzyu.github.io/AlexZyuNotebook/ 上。

如何配置？

    1. 使用git clone git@github.com:AlexZyu/AlexZyuNotebook.git将本地文件夹与远程Github仓库绑定。
    1. 在根目录创建deploy.sh文件，写入以下内容：
#!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
npm run docs:build

# 进入生成的文件夹
cd docs/.vuepress/dist

git init
git add -A
git commit -m 'deploy'

git push -f git@github.com:AlexZyu/AlexZyuNotebook.git master #修改为具体的sh地址

cd -
    1. 修改config.js 添加
module.exports = {
	base: "/AlexZyuNotebook/" //base根据Github上的仓库名确定
	}
    1. 修改package.json 添加
  "scripts": {
    "deploy": "bash deploy.sh"
	}
    1. 使用Git Bash输入yarn deploy即可。
    1. 打开open in new windowhttps://alexzyu.github.io/AlexZyuNotebook/。
