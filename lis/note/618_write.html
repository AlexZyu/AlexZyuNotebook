# 检索语言

  1. 概念组配：建立在概念逻辑关系的基础上，以概念的分析与综合为手段，以揭示概念的本质为目标，利用检索语言中已有的若干概念，组合起来表达一个新的专指概念的方法。
  2. 字面组配：根据构词法原理，将一个具有专指性的复杂概念按其字面构成分拆为两个或两个以上的单词，标引时再将两个词拼接起来，表示一个专指、复杂概念的方法。
  • 先组式语言
    • 标题词语言

    先组定组

  • 后组式语言
    • 单元词语言
    • 叙词语言
    • 关键词语言
  • 主题检索语言：以直接表达文献内容概念的语词作为检索标识，以字顺排列为主要检索方式，并通过参照系统表示词间关系的标引和检索语言。
    • 叙词：以表示单元概念的规范化语词为基础，以概念组配为基本原理，。。对文献主题进行描述的后组式检索语言。
    • 标题词
    • 关键词
    • 单元词
  • 未经过规范化处理
    • 关键词
  • 经过规范化处理
    • 单元词
    • 标题词
    • 叙词

定义：分类主题一体化词表又称分类主题一体化检索语言，是指在同一检索系统中，将分类表和主题词表的术语、参照、标识和索引实施统一的控制，使其有机地结合起来，并同时具有分类表和主题词表的标引和检索的功能。

对比：

  • 分类法：按照学科对事物进行集中，具有良好的分类/等级检索能力。具有较好的系统性。
  • 主题法：按照事物主题来集中文献或信息，有较好的直接性和专指性。

能够结合起的原因？原理？

原理上的 相似性 一致性和结构上的差异性。

  1. 有相同的原理基础，即概念逻辑、术语学和知识分类原理
  2. 所表达的主题概念的标识在本质相同
  3. 处理对象都是语义单元，表达的都是主题概念

# 信息标引

信息标引是在分析文献主题的基础上，用某种检索语言对文献主题或其它有意义的特征标识 对文献赋予检索标识的过程。它是文献存储与检索的一种文献处理过程。

简单来说，标引就是依据检索语言，赋予检索标识的过程。

信息检索的原理：根据特定的检索需求，在数据库中将检索条件与检索标识进行异同比较，最终输出其中相符或部分相符的信息。

信息标引是指在分析文献主题的基础上，利用某种特定的检索语言对文献的主题内容或其它特征标识出来，它是赋予文献标识检索

  • 广义的信息是指一切由人或自然界所传递出来的符号、信号，是实物存在的客观属性。
  • 狭义的信息是指经过搜集、组织、加工、存储以供检索的文献、事实或数据。
  • 广义的信息就包括：
    • 自然信息、社会信息
    • 自在、自得、再生
  • 标引质量控制
    • 标引深度
    • 专指度
    • 一致性

标引质量的控制：标引深度->专指度->一致性

步骤：主题分析->概念转换

  • 主题分析：中心思想

信息标引可分为主题标引和分类标引。主题标引面向特性检索，分类标引面向族性检索。

  • 分类标引：依据一定的分类检索语言，对文献内容的学科性质和部分外部形式特征进行分析、标引并赋予分类检索标识的过程。
  • 主题标引：依据一定的主题词表，对文献的内容特征进行主题分析，再赋予文献语词标识的过程。

依据什么？先分析什么？后赋予什么标识？

自动分类：由计算机自动提取信息的特征项，将信息按内容归到一个或多个类别中。分为归类和聚类，区别在于是否提前确定分类体系。

分表标引：根据一定的分类检索语言，对文献的学科性质和其它具有检索意义的形式特征进行分析、选择、赋予分类检索标识的过程。
