# 第三章 信息分布【已结束】

基础定律：马太效应产生的核心趋势和集中取向。

四大主题：

  • 文献
    • 布拉福德定律：将文献的在某学科的出版数量按递减顺序排序，可以找到一个1：a：a2的分区。
  • 内容
    • 齐夫定律：将文献内出现的词汇按照频次从高到低排序，并标注其序号，能找到c=f*r，其中f是频次，r是等级序号。
  • 作者
    • 洛特卡定律（1926, Lotkas）：在论文数X和作者数Yx之间，存在着下列一种关系：XyYx = C
    • 普赖斯定律：平方根定律，在某一特定领域中，全部论文的半数系由该领域中全部作者的平方根的那些人撰写的。
  • 时间
    • 普赖斯曲线：科学文献增长与时间呈指数函数关系。这是社会信息流的传播和影响的结果。
    • 生长曲线：科学文献信息的增长趋近于生物的生长曲线。区分出诞生、成熟、衰退期。
    • 逐渐过时率
      • 半衰期：某学科尚在利用的一半的文献是在多少年内产生的。
      • 普赖斯指数：某一学科领域内，对发表年限不超过5年的文献的引用次数与总引用次数之比值。

# Lotkas Law(Invers square law)

信息管理领域内，可以理解为文献的作者分布情况；而在更广义的人类学范围内，可以理解为不同个体对社会的贡献情况。有这种差异的机制主要原因是马太效应。

  • 洛特卡定律的发现并未考虑合作者在文献分布规律中的作用。
  • 随着科学技术的发展，合作著述已成为今天科学劳动的主要形式，因此原始的洛特卡定律必须向合作论文研究的更高境界发展。

洛特卡定律实际上是科学论文在作者上的集中与分散的分布现象，具有两个基本特点：

  1. 采用频次排序，按作者出现频次排序，而非作者的论文数量。
  2. 论文在作者上的集中与分散程度仅限于平方反比关系。

提示🎗️

一般表达式（广义洛特卡定律）： f(x) = C / xn

n的数值并非简单地为2.0，每个领域不同，大概为(1.2, 3.5)（Vlachy, 1976）。

此处是不是可以作为毕业论文，探索现在的情况？或者，对过去学者利用样本得出的研究情况作验证和检验？

如果要做，那么时间跨度要大，样本容量要大。

# 普赖斯曲线

科学文献增长与时间呈指数函数关系： F(t) = a * ebt (a>0, b>0)

其中：

  • t：时间，单位：年
  • a：t=0时文献的初始量
  • e：自然对数的底
  • b：时间常数，即持续增长率

# 普赖斯指数

对文献按年代被引证的次数进行分析，可以得到某一门类或某一学科领域的文献老化速度最可靠的指标。

# 信息源

信息源：在一切人类活动中产生的成果和原始记录，及其经过加工、整理后形成的原始记录都成为人们获取信息的源泉，简称信息源。

联合国定义：满足人们信息需求的 个人为满足其信息需要（信息素养：信息意识、信息能力、信息知识、信息伦理）而获得信息的来源。

信息源->倾向于解释为物质载体？ ----> 广义信息源包括：信息载体+信息机构

信息源的基本属性：

  • 客观性：载体客观、内容客观
  • 可传递性：发送、接收
  • 可激活性：直接被感知和利用

信息源的特征：

  • 积累性
  • 再生性
  • 复杂性
  • 共享性

信息的特征？

  • 普遍、客观 --> 基本属性 --> 特征
  • 广延、无限 --> 特征 --> 特征
  • 传递性 --> 基本属性 --> 特征
  • 独立性
  • 相对性 --> 基本属性
  • 共享性 --> 特征
  • 不可变化和组合
  • 时效性

# 记录型信息源

存在于各种传统介质和现代介质中的知识信息。

特点：

  • 是信息传播系统，是信息存在的 主要 基本方式，是主要的信息管理对象
  • 一般通过正式渠道流通，也通过新闻系统等自然渠道流通 传递

分类：

  • 按物质载体分为
    • 印刷：纸本 最多、最广 方便 但难以管理
    • 微缩：感光材料、介质
    • 机读：二进制，CD、磁盘、硬盘等存储
    • 网络：电子形式，网络传播、发展最快，非纸质载体，网上存取
  • 按加工程度分为
    • 一次：原始信息、记录
    • 二次：对一次加工、整理而成，是一次的线索
    • 三次：对一、二次的加工、凝练、概括，信息价值最高，包括综述、评论等
  • 按流通形式分为
    • 图书
    • 期刊
    • 会议资料
    • 专门报告
    • 专利
    • 政府出版物
    • 学位论文
    • 新闻报纸

# 分类法

  • 《信息检索》定义
    • 将许多类目根据一定的原则组织起来，通过标记符号（分类号）来代表各级类目和固定其先后次序的分类体系。是直接体现知识分类的概念系统，是对概括文献信息内容及某些外表特征的概念运用逻辑的方法进行系统排列而构成的。
  • 《信息管理基础》定义
    • 根据信息资源的内容属性和其他特征，将资源分门别类地、系统地组织和揭示的方法。它用分类号来表达各种概念，将各种概念按学科性质进行分类和系统排列。
