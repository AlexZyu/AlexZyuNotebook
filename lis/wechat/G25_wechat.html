# 图书馆学G25

# 001 信息组织

  • 信息组织是根据信息资源检索的需要，以文本及各种类型的信息源为对象，通过对信息资源内容特征等的分析、选择、标引、处理，使其成为有序化集合的活动。
  • 信息组织是为信息检索服务的，信息检索是信息组织理论的逻辑起点，开放互联及多学科融合的信息利用是信息组织的方向。

# 002 分类法

  • 定义：将表示各种知识领域和类目按照知识分类原理进行系统排列，并以代表类目的分类号作为文献主题标识的一类情报检索语言。
  • 一般来讲，文献分类以文献的内容特征作为主要分类标准，以形式特征为辅助标准。
    • 信息的分类组织是构建有序化信息系统最常用的方法。从网络环境下信息组织需要来看，分类方法依然是信息组织的主流方法，应用前景广阔。
  • 举例：中国图书馆分类法（China Library Classification,CLC）和杜威十进分类法（Dewey Decimal Classification,DDC）

# 003 主题法

  • 定义：直接以表示文献主题的语词作为标识，提供字顺检索途径，并主要采用参照系统揭示词间关系的标引和检索文献的方法。
  • 主题法完全建立在自然语言的基础上，很好地满足了特性检索的需要，而且与计算机信息检索技术的发展相适应。
  • 类型：标题法、单元词法、叙词法、关键词法

# 004 先组式语言

  • 先组式语言(pre-coordinate index language)与“后组式语言”相对。在检索实施前已将检索词事先组配好的一种检索语言。用户只能用这种已经固定好的检索词组形式去完成检索。这种语言有较好的直接性和专指性，但灵活度差。标题词语言（标题法）属于先组式语言。

# 005 后组式语言

  • “后组式语言”(post-coordinate index language)与“先组式语言”相对。在检索实施前检索词未事先组配好，而是以单元词等形式出现的一种检索语言。用户在检索时将它们临时组配起来，表达一定的概念并完成信息检索。后组式语言比较灵活。单元词语言（元词法）和叙词语言（叙词法）等属于后组式语言。

# 006 等级列举式分类法

  • 等级列举式分类法是文献分类法的一种类型。具有严密的类目等级结构和以详尽方式列举的代表信息与知识概念的类目。强调按知识的系统性去组织文献，便于读者按学科、专业系统检索图书资料，也便于图书馆按学科体系组织馆藏。但由于类表结构固化，不便修订与增补，其容纳性与适应性受到限制。
  • 也称列举式分类法、枚举分类法、等级分类法、体系分类、系统分类法、展开式分类法、层次分类法。

# 007 分面组配式分类法

  • 分面组配式分类法是根据概念的分析与综合的原理，将概括文献、信息、事物的主题概念组成”知识大纲—分面—亚面—类目“的结构，按一定的规则、通过各个方面内类目之间的组合来表达文献主题的一种分类法。
  • 也称组配式分类法、分面分类法、分析—综合式分类法。
  • 分面分析、分面引用次序、分面标记制度，是分面组配式分类法的核心理论和关键技术。

# 008 基本部类

  • 文献分类表中为便于各种类目的展开而对人类全部知识与客观事物所做的最基本、最概括的划分。

# 009 基本大类

  • 在基本部类基础上，根据学科发展和文献出版情况，在分类表中首先划分出来的第一级类目，整个分类表在此基础上层层展开。基本大类一般都是传统的学科领域，并没有一定的定例。

# 010 类目级别

  • 代表该类目在分类体系中的等级（划分的层次）、显示类目间的等级关系。类级在印刷版或文本界面中使用排版的缩格和字体表示，在数据库中用特定的字段标记。

# 011 类列

  • 由一个上位类区分出来的一组下位类互称同位类，一组同位类总称为一个类列。
  • 按照某种标准划分某类目而产生的一组并列的下位类的总称。这些类目相互排斥，整个类列应该穷尽上位类的外延。

# 012 简表

  • 分类法简表也称基本类目表，是由分类法的一级大类（基本大类）进一步区分出来的二、三级类目所组成（部分知识领域可能包括四级类目），是分类法的类目体系框架，分类法基本知识结构与划分规则等可以通过基本类目表清晰地展现出来。

# 013 复分表

  • 为增强类目细分程度和节省篇幅，把分类表主表中采用同样标准区分的子目汇集而成的表。可分为适用于整个分类表的通用复分表和仅适用于某一大类或专门学科的专用复分表。分类时按一定规则将复分号附加于基本类号之后，形成新的、专指性更强的分类号。

# 014 通用复分表

  • 对分类表中各类目都适用的复分表。通常附于文献分类表主表之后。
  • 也称分类法附表、分类法辅助表。

# 015 专类复分表

  • 只供特定类目细分时使用的复分表。一般列于主表中适用的类目下。
  • 又称专用复分表

# 016 仿分

  • 对体系分类表未列出的类目，仿照相似类目的一组下位类目进一步细分的方法。即利用某一类的子目类作为进一步细分的依据。

# 017 类目划分

  • 类目划分是选用一定的分类标准，对一个较宽泛的上位概念进行分组，形成一组平行的类目，这组平行的类目互称同位类，它们都具有其上位类共同的内涵，同时又增加了分类标准所限定的内涵。
  • 类目划分是建立分类法整体知识结构和局部知识结构的过程，决定着文献、信息聚类的方向和集中的程度，对文献信息检索效率有着重大影响。

# 018 引用次序

  • 引用次序也称分类标准的使用次序，是等级体系分类法类目划分时使用分类标准的先后次序。
  • 一个类在连续划分的过程中，需要分别使用该类事物的若干属性作为分类标准。这些分类标准的使用次序，决定着文献、信息聚合成类的次序和按某种属性集中的程度，决定着分类法局部的类目体系结构。
  • 越往后使用的属性聚类的能力越弱，因此分类标准使用的选择，对分类法的性能有重大的影响。

# 019 标记符号

  • 分类号的标记符号，也称分类号、类号，是类目的代号。标记符号是现代文献、信息分类法重要的、不可少的组成部分。分类法全部类目的标记符合及其构成的整体联系，就是分类法的标记系统。

# 020 标记制度

  • 分类法的标记制度也称编号制度、配号制度，是编制分类标记的根本方法，决定着分类标记系统的类系及性能。
  • 其中最常使用的是层累标记制、顺序标记制、混合标记制和分面标记制，不同的标记制度产生不同的标记符号类型。

# 021 层累标记制

  • 层累标记制又称等级标记制、展开标记制、树形标记制。是类号的位数与类目的等级相对应的编号制度，一般用一位符号标记一级类，用两位符号标记二级类、用三位符号标记三级类，以此类推层层累加，与类目的划分层次相同步。
  • 其优点是能够解释类目的等级关系、有很强的表达性、便于扩检或缩检。
  • 其缺点是随着类目的逐级展开号码将越来越差，影响简短性；同位类数量受限，或需变通处理。

# 022 顺序标记制

  • 顺序标记制是在配号时不管类目的等级，只按类目排列顺序分配号码的标记制度。
  • 其优点是配号简单，号码较简短、号码利用率高。
  • 其缺点是表达性差，不能体现类目之间的等级关系，不便于扩检或缩检，对计算机信处理的适应性也差。
  • 如《美国国会图书馆分类法》、《概略分类体系》等。

# 023 混合标记制

  • 混合标记制是把层类制与顺序制相结合的一种标记制度，包括层累—顺序制和顺序—层类制两种基本类型。
  • 较多采用的是在层累制的基础上有限、灵活地加入顺序制配号法，以缩短号码的长度或局部增加号码的容量。
  • 如《科图法》。

# 024 标记系统

  • 分类法的标记系统是与分类法的基本类型相适应的，为分类法类目体系构建、文献标引、文献组织、文献检索服务。

# 025 号码扩充技术

  • 由于知识体系自身的复杂性、事物本身的多样性，在编制分类法类目体系时会出现同位类极不平衡的现象，有的类目可能只有一、二个下位类，有的却多达几十个甚至上百个下位类。当原号码系统按常规不能满足类目配号时，就必须根据类目实际展开的情况和今后可能的需要,灵活地解决号码扩充问题。号码的扩充也要有一定规律性，相同或相似的情况下尽量使用相同的扩号方法。

# 026 借号法

  • 所谓的借号，是指同位类展开时比常规列类多出一、二个类目(包括不占用为了今后扩充需要预留的号码),而其前面的同位类展开后子目较少，剩余了若干号码(除去为了今后扩充需要预留的号码和进行复分预留的号码)，这时就借用前面类目的下位类号为该类目配号，这种情况号码的长度已经与类级不相对应。选择借号的类目一般是重要性较低的类目。

# 027 八分法

  • 八分法也称扩九法，是同位类较多（或预计发展较多）但不超过18个时常用的扩充编号技术。基本编号方法是：使用前八个号码为前八同位类配号，将剩余的"9"扩成双位，为其他同位类配号。在八分法扩号的基础上还可以进行第二层的扩九，但这样同级类目的号码位数差距太大，一般很少使用

# 028 双位法

  • 双位法也称百分法，是用于同位类数量相当多时的一种扩号技术。基本的编号方法是：1—9本身不使用，每个号码直接扩充为双位号，为同位类配号，这样在同一个类列中号码的位数与类目的等级是相对应的。双位扩号法在实际使用中也有各种变通，可以根据需要只选择部分数字扩充为双位号码，也相当于八分法在同一层的多次使用。实际上，八分法是双位法的一种特例

# 029 空号技术

  • 在分类标记系统设计和实际配号中，为了给类目扩充或修订留有余地，或为了编号的逻辑性，或为了编号的对应性等目的，大量使用空号技术。
  • 常用的空号方法有：
    • 间隔空号：当号码比较充裕时、号码的预留难以预测时使用。
    • 对应性空号：为达到对应编号的目的，为性质相同的类列中但没有列举的类目配以对应性空号。
    • 逻辑性空号：当列类时按逻辑性的原则应为某事物设立某类目，但由于当时文献保证不足等原因没有列类，则为该主题应占据的位置进行逻辑性空号。
    • 尾部空号：这是最常用的空号法，编号时类列的前面类目依次顺序编号，把“其他”或“XX应用”置于类列最后，用“9”配号，其余尾部号码为空号。
    • 预测性空号：在编表时根据类目今后可能展开的情况，有预见地在一定位置预留空号。

# 030 查全率

  • 查全率（recall ratio, R）是衡量某一信息检索系统从特定文献信息集合中检出相关文献信息成功的一项指标。
  • 查全率（R）= 被检出相关的文献信息量 / 相关文献信息总量
  • 查全率与漏检率之和为1。

# 031 图书馆学五定律

  • 第一定律：书是为了用的（Books are for use）
  • 第二定律：每个读者有其书（Every reader has his book）
  • 第三定律：每本书有其读者（Every book has its reader）
  • 第四定律：节省读者的时间（Save the time of the reader）
  • 第五定律：图书馆是一个生长着的有机体（A library is a growing organism）

# 032 采准率

  • 采准率：在规定的学科、主题或层次范围内，用来测量馆藏中切题文献收集的针对性的一种尺度，即该范围内馆藏中切题文献数量在当时全部馆藏文献中所占的比例。可用公式表达为：采准率=馆藏中切题文献量/馆藏文献总量×100%

# 033 组配

  • 在编制检索语言词典或标引构成检索提问式时，把两个或两个以上的标识或符号组织起来的、成一个较大语法单位以在标引或检索中准确表达某一主题。

# 034 文献老化

  • 文献在产生或出版以后，随着其“年龄”的增长，受到各种因素的影响其内容的价值逐渐降低，从而导致其利用率越来越低的现象。

# 035 半衰期

  • 半衰期（half life）是表征文献老化速度的指标。分为历时半衰期和共时半衰期。
  • 历时半衰期指某学科（专业）已发表的文献中有一半已不被使用的时间。
  • 共时半衰期也称中值引文年龄，指某学科（专业）现时尚在利用的全部文献中较新的一半发表的时间。

# 036 类系

  • 类系：连续划分的一系列具有从属关系的类目称为一个类系。

# 038 回溯标记制

回溯标记制：结合倒排类表，直接将标志类表中在前分面的号码做分面指示符，利用在后面号码配置时预先留下的空号进行组配，通过标记中的字顺颠倒表示新分面的引进，显示类目的组配结构。

# 039 普赖斯指数

  • 普赖斯指数是衡量文献老化速度的指标之一，即在某一个知识领域内，年限不超过5年的引文数量与引文总量之比。由普赖斯提出，用以量度文献的老化速度和程度。
  • 其计算公式为：Pr=出版年限不超过5年的被引文献数量/被引文献总量×100%。式中，Pr为普赖斯指数。
  • 一般来讲，某一学科领域文献的普赖斯指数越大，其半衰期就越短，其文献老化速度也就越快。

# 040 剩余有益性指标

  • 1970年代，英国信息学家布鲁克斯（B. C. Brookes）引进期刊有益性的概念，将期刊的剩余有益性作为评价其老化的指标。
  • 期刊有益性是指某一年份某一期刊被用户利用的文章数。
  • 期刊剩余有益性是指经过若干年后，期刊仍保留着的有益性，它是期刊老化程度的一种度量。
  • 采用期刊剩余有益性指标衡量老化程度，假定了所收录的每一种期刊的有益性是以相同的速度减少的，并且只对于满足一定类型和内容的信息需要的具体期刊才是适用的。

# 041 引文分析法

  • 引文分析法是将所研究的学科领域在一定时间之内的全部文献信息（尤其是科学论文）收集起来，利用数学中概率论及数理统计的方法，通过统计每篇论文所附的引文的发表时间及其被引用时间等数据来研究该领域文献老化的情况，以揭示其数量特征和内在规律。

# 042 文献流通率

  • 单位时间内，用于公开借阅的馆藏文献被读者借阅的数量占馆藏可借阅文献总量的百分比。反映图书馆馆藏文献的利用情况和读者的阅读需求程度，是衡量馆藏文献质量和流通工作的重要尺度，也是研究读者的重要素材。

# 043 叙词

  • 叙词（descriptor），我国多称主题词，是取自自然语言，经过规范化处理的，以基本概念为基础的表达文献主题的词或词组。

# 044 非叙词

  • 叙词的同义词或准同义词，在叙词表中规定不能用于标引和检索，只起指引作用，指向对应叙词的词。

# 045 概念组配

  • 建立在概念逻辑关系的基础上，以概念的分析与综合为手段，以揭示概念的本质为目标，利用检索语言中已有的若干概念，组合起来表达一个新的专指概念的方法。

# 046 字面组配

  • 根据构词法原理，将一个具有专指性的复杂概念按其字面构成分拆为两个或两个以上的单词，标引时再将两个词拼接起来，表示一个专指、复杂概念的方法。

# 047 先组词

  • 以复合概念的语词形式收入词表的词。

# 048 后组词

  • 以单一概念的语词形式收入词表的词。

# 049 族首词

  • 族首词是一组具有等级关系的词中外延最广的词，通常是排检词族索引的依据。

# 050 单元词

  • 单元词（uniterm）是指用来表示文献主题的、最基本的、在概念上不能再分解的词。

# 051 叙词款目

  • 叙词款目是字顺表的构成单元。

# 052 轮排索引

  • 又称轮排表，是将叙词按其词素的字顺轮流排列到检索入口，使含有相同词素的词集中在一起的索引。

# 053 交叉组配

  • 是指使用两个或多个具有交叉关系的叙词进行组配。

# 054 方面组配

  • 是指将表示某一事物的叙词与表示事物方面（部分、属性、状态、过程、条件、关系等方面）的叙词进行组配。

# 055 范畴索引

  • 又称分类索引、范畴表，是按叙词表全部叙词和非正式叙词所属学科或范畴编制的词汇分类索引。

# 056 中国分类主题词表

  • 《中国分类主题词表》是在《中图法》编委会的主持下，从1987年开始由全国40个图书情报单位共同参加编制，1994年出版的一部大型文献标引工具书。它是在《中图法》第三版（包括《资料法》第三版）和《汉语主题词表》（以下简称《汉表》）的基础上，为实现分类主题一体化标引，为机助标引、自动标引提供条件，降低标引难度，提高检索效率和标引工作效率，编制而成的分类检索语言和主题检索语言兼容互换的工具。

# 057 参照系统

  • 根据参照法所编制的一般参照、单纯参照、相关参照等的合称。旨在对信息进行组织时，把相互关联的信息标目或款目联系起来，指引用户从一条标目或款目参考和检索其他有关标目或款目。

# 058 零增长理论

  • 零增长理论（Zero-growth Theory），又称稳定状态理论（Steady State Theory），其核心内容是要求建立有限规模的图书馆，在图书馆达到一个可靠的目标(馆藏量、功能等指标)之后，剔除文献的速度应当等同于采购的速度，即图书馆新购进的资料只是对准备剔除的资料的相当补偿，馆藏的实际增长速度为零，因而使图书馆收藏的文献总量保持一种稳定状态。

# 059 信息标引

  • 信息标引，又称文献标引，是根据文献的特征，赋予文献检索标识的过程。信息标引所根据的文献特征以内容特征为主，外部（形式特征）为辅。
  • 信息标引包括主题分析和转换标识两个过程。

# 060 分类标引

  • 又称文献分类或信息分类，是依据特定分类语言，对信息资源的内容特征进行分析、判断、选择并赋予文献分类标识的过程。

# 061 主题标引

  • 主题标引，是依据特定的主题语言，信息资源的内容特征提取信息资源主题，赋予信息资源词语标识的过程。

# 062 深标引

  • 这是赋予文献较多标识，详细揭示文献内容的标引方式。全面标引必然是一种深标引。深标引主要适用于计算机检索系统和学术论文的标引。

# 063 浅标引

  • 这是赋予文献较少标识，粗略揭示文献内容的标引方式。整体标引必然是一种浅标引。浅标引主要适用于手工检索系统和图书的标引。

# 064 自动分类

  • 自动分类（automatic categorization）是指在给定的分类下，根据文本的内容自动地确定与文本关联的类别。从事数学角度而言，分类的实质是一个映射的过程，它将待标明类别的文本根据映射规则映射到已有的类别中，该映射可以是一对一映射，也可以是一对多映射。

# 065 自然语言检索系统

  • 自然语言检索系统是指对文献作者或文摘提要的编写者原来使用的语言进行一定的序化组织、处理并提供自然语言检索接口供用户查询使用的检索系统。

# 066 全文检索系统

  • 全文检索系统指具有全文数据库，具备布尔逻辑检索、位置检索、字符串检索、截词检索等检索功能，允许用户以自然语言检索，直接获得原文中的有关章、节、段、句等信息的一类检索系统。

# 067 后控制词表

  • 后控制词表是利用受控语言的基本原理和方法编制的自然语言检索用词表，它主要是对自然语言中大量存在的等同关系、等级关系和大部分相关关系进行控制和揭示，具有自学习功能，可根据检索的需要将新概念和新术语及时地加入词表中，因此，后控词表是一个动态词表。

# 068 自然语言

  • 从信息检索角度来理解，自然语言是指文献作者所使用的书面用语，在信息检索中包括关键词、自由词和出现在文献题目、摘要、正文或参考文献中的具有一定实质意义的词语。

# 069 受控语言

  • 通过对文法、词汇加以限制而形成的某一学科领域的自然语言的子集。目的是减少歧义等现象的出现。

# 070 自由标引

  • 自由标引即人工关键词标引，是由标引人员在对文献内容进行分析之后，按一定规则，将文献原文所用的，能描述文献内容主题的那些具有关键性意义的词抽出，不加规范或只做少量规范化处理，按字顺排列，以提供检索途径的方法。
  • 就其实质而言，是一种在信息检索中利用自然语言的方法。

# 071 自动标引

  • 自动标引，指利用计算机从各种文献中自动提取相关标识引导的过程。
  • “广义”的自动标引包括自动主题标引和自动分类标引。
  • “狭义”的自动标引是指自动主题标引，包括自动抽词标引和自动赋词标引两种方式。

# 072 停用词

  • 停用词（stop word）指在计算机对文献进行抽词标引的过程中所遇到的不能做标引词或检索词使用的词语。包括介词、冠词、代词、连接词、某些形容词和副词，以及一部分太泛指和使用频率过高的实词（普通名词）。它们既不能反映文献的实际内容，也不能用来检索文献。

# 073 截词检索

  • 截词检索（truncation search）是一种允许检索者使用一个词的局部（如词干、词根）作为检索词进行匹配并输出命中文献的一种检索词局部匹配技术。

# 074 互见标引

  • 互见标引是从不同角度多次揭示同一文献内容的标引方式，即对已经从一个角度或学科标引了的文献内容，再从其他角度或学科出发予以标引。这也是一种补充 标引，其目的是从不同角度或学科揭示同一文献内容。
  • 互见标引主要用于分类标引，尤其是类号不能组配轮排的分类标引，以便提供检索同一文献的不同途径。

# 075 靠词标引

  • 靠词标引（homoionym indexing）是采用与主题概念含义较相近的类目或主题词做文献标引词的标引方法。
  • 当分类表或主题词表中没有与文献主题概念对 应的类目或主题词，旦既无法通过组配获得此概念，又不能采用上位标引时，可采用这种标引方法。这是一种迫不得已的处置办法，应慎重使用，并认真做好标引记录。

# 076 增词标引

  • 增词标引（adding words indexing）是采用临时增补的主题词做文献标引词的标引方式。当主题词表中没有与文献主题概念相对应的主题词，且无法采用上位标引、组配标引或靠词标引，确实需要增加一个新主题词做文献的标引词时，可采用这种标引方法，但只限于相关文献数量较多、新增词在标引中使用频率较高的情况。

# 077 叙词词库

  • 叙词词库是将多部叙词表的词汇汇集存于计算机，从严格意义上说，它是叙词表库。
  • 叙词词库兼具词汇转换、宏观词表与微观词表兼容、生成集成词表等功能。

# 078 中介词典

  • 中介词典是一种转换方法或交换语言，是指利用中介词典进行多词表间的转换，它能将任何一个信息系统中标引文献时给出的标引词或分类号转换成任何其它信息系统中的概念等价词。
  • 中介词典不是一个专供特定信息检索系统标引和检索用的词表。

# 079 分面叙词表

  • 分面叙词表是一种分类主题一体化词表，是最经典、影响最大的分类主题一体化检索语言。分面叙词表的基本结构包括分类表和叙词表两大部分，有些还编有轮排索引、化学式索引或专有叙词索引等。
  • 分面叙词表是把分面分类法或半分面分类法的类名叙词化，叙词严格按照分面分类法聚类和配号，做到叙词和类号一一对应，实现类与主题等值概念的转换和对应，使分类表和叙词表完全融为一体的词表。
  • 其代表为：艾奇逊等的《分面叙词表》。

# 080 叙表式索引

  • 叙表式索引的特点是把分类表的字顺索引款目改造为叙词款目，在索引款目中增加类名的用、代、属、分参等参照及各自相应的分类号，以增强词间关系（即主题概念之间的关系）的显示。

# 081 分类表—叙词表对照索引

  • 这种对照索引是分类检索语言和主题检索语言兼容、互换的工具，通常用作信息检索查词选类的辅助工具，便于分类标引数据和主题标引数据之间的转换。
  • 如果在这种对照索引的分类号（以及类名）和叙词下加上原有的注释项和参照项，就可以同时用于分类标引和主题标引。

# 082 主题分析

  • 主题分析，一般是指标引过程分析文献内容和提取主题概念的环节。
  • 具体地说，主题分析是通过审读文献而了解文献内容及其表现形式，选择内容单元而确定文献主题数量，概括单元内容而形成主题概念，分析主题的结构和类型，并在必要时判断主题的学科专业属性的一种智力劳动。

# 083 分类法使用本

  • 分类法的使用本，是指具体单位根据自己的文献情况和用户需要，在允许的范围内，对通用分类法作适当的调整、补充、说明后，确定下来作为分类标引最后依据的本子。

# 084 图书改编

  • 如果作为分类标列依据的分类法经过修订出了新的版本，或者有的单位想放弃原来使用的分类法，采用一种新分类法，就需要用新的分类法版本或新的分类法对已经分类标列过的图书重新进行分类标号，这就是图书改编问题。

# 085 书次号

  • 用分类法对图书进行分类标引的结果，必然是绝大多数类目下分别聚集了若千种或许多种图书，亦即多种图书拥有同一个类号。为满足检索和管理的需要，还必须对同类书进行区分，实现同类不同种的图书号码个别化。对同类书中特定图书的个别化编号，就是同类书的区分号，简称为书次号。

# 086 辅助区分号

  • 同一类图书，经过著者号或种次号区分后，相当一部分能够实现号码个别化。但是，还会有一部分没有实现号码个别化。因此，作进一步区分用的号码称为辅助区分号。
  • 图书馆的分类索书号一般是由分类号、书次号和辅助区分号组成。

# 087 自然语言处理

  • 自然语言处理 (Natural Language Processing，简称 NLP) 就是利用计算机为工具对人类特有的书面形式和口头形式的自然语言的信息进行各种类型处理和 加工的技术，这种技术现在已经形成一门专门的边缘性交叉性学科，它涉及语言 学、数学和计算机科学等多个学科。

# 088 自然语言标引

  • 自然语言标引是指采用原文中的信息作为标引源，从中选取能够有效表征信息内容的特征词。

# 089 自然语言检索

  • 所谓自然语言检索，目前还没有明确的定义，我们可以从三个方面来界定它：
  • 从检索语言的角度来讲，自然语言检索就是指在为文本信息提供检索标识时，使用文献作者、文摘编写者原来所用的语词或标引人员自拟的语词，而非取自受控词表中的语词；
  • 从技术层面上来理解，就是将自然语言处理技术应用于信息检索系统的信息组织、标引与输出；
  • 从用户层面而言，就是用自然语言作为提问输入的检索方式。

# 090 全文检索

  • 全文检索 (full-text retrieval) ，是指以全部文本信息作为检索对象的一种信息检索技术。全文检索技术是一种面向全文、提供全文的新型检索技术。可以使用原文中任何一个有实际意义的字、词作为检索入口，而且得到的检索结果是源文献而不是文献线索。

# 091 位置检索

  • 位置检索，又称相邻度检索，是指通过对两个或多个词之间的位置关系所进行的限定检索。
  • 比如要求被检索的词汇在原文中以一定顺序出现，限制两词之间相隔的词汇或字符个数，要求两词出现于同一自然句、同一段或同一篇文章中等等。
  • 位置检索是全文检索系统中特有的检索技术，最能体现全文检索系统的优势。

# 092 字索引

  • 字索引也称为字表法，是全文索引的一种，它以单个汉字为基本索引单元。字表法是对每个单字的出现位置进行索引，并根据单字的位置信息进行计算匹配的文本检索方法。
  • 字表法索引库的主要部分是每个字的字表信息。

# 093 词索引

  • 以词或词组为索引单元的索引也称为词表法。
  • 词表法是以能表达一定意义的词为基本检索单位，并根据词的出现位置进行索引和检索的文本检索方法。以词为单位建立索引时，一般需要对源文档进行切分处理，然后对切分后的文档词条进行统计，记录每一个出现的词条及其出现的位置，这种方法需要分词词典的辅助。

# 094 N-Gram法索引

  • N-Gram 即采用n元语法的切分统计方法来自动发现相关概念，然后对提取出来的词建立索引。根据中文构词的特点，以二字词和三字词居多，因此中文全文检索系统中常用的为二元（2-Gram) 及三元 (3-Gram) 语法。
  • N-Gram法无需任何词典支持，对输入文本所需的先验知识少，无需进行分词处理，将文本分解为索引项集合非常容易；但会产生非常大的数据冗余，查准率也比较差。

# 095 搜索引擎

  • 对网络信息资源进行搜集整理并提供信息查询服务的系统，包括信息搜集、信息整理和用户查询三部分。
  • 搜索引擎普遍采用了全文搜索技术。

# 096 简单提问式输入检索

  • 以自然语言理解技术为基础的新一代搜索引擎，我们称之为智能搜索引擎。
  • 现在，已经有越来越多的搜索引擎宣布支持自然语言搜索特性，这种特性多指系统在“用户一检索接口”层面上支持用户以简单自然语言提问的方式输入检索式。目前搜索引擎所宣称的自然语言检索功能多指这种简单提问式检索技术。

# 097 问答系统

  • 问答系统（QA系统）是根据用户的自然语言提问，从文本库或数据库中提取相关信息并用自然语言输出答案的计算机应用程序。
  • 一般包括三个主要模块，即问题分析模块、信息检索模块和答案抽取模块。

# 098 Ask.com

  • Ask.com通过将用户提问转化为系统已知问题然后对已知问题进行解答，以求降低对自然语言理解技术的依赖性，是一种基于自然语言形式的自然语言语句查询模式。Ask.com是这种检索模式最杰出的代表，它开创了智能搜索引擎查询的新模式，开辟了一条实现自然语言检索的"捷径"。但是，我们也应该认识到Ask.com 的核心技术——问题与答案数据库相对于人类知识而言是非常局限的。

# 099 AnswerBus

  • AnswerBus是一个基于句子层面、面向开放领域的自然语言问答系统。
  • AnswerBus接受用户以英语、法语、德语、西班牙语、意大利语五种语言的 自然语言提问，并根据问题的类型和内容动态选择和调用Google、Yahoo!、WiseNut、AltaVista、Yahoo!News这五个搜索引擎和主题目录中最佳的23个用于检索，获取各搜索工具中最靠前的命中文档作为参考答案。

# 100 后控制模式

  • 后控制模式，即“标引不控制+检索控制”模式。
  • 这种模式是在标引（输入）阶段使用自然语言，不对标引进行严格控制，而在检索（输出）阶段才对检索词进行控制的自然语言检索优化技术。
  • 实践表明，后控制模式是提高自然语言检索系统检索效率的有效措施。尤其在网络检索系统中，后控制技术更是有了全新的功用。

# 101 停用词典

  • 停用词典收录所有对检索无意义和高频率的虚词、介词及增加系统资源开销的插入语等性质的词和短语。
  • 在用户输入检索式后，系统自动调用停用词典对检索式进行分析，排除无意义的字或词或短语，然后将经停用词典过滤控制优后的检索式交检索系统作进一步优化处理。

# 102 截词词典

  • 截词词典——有效检索词词形控制。
  • 截词检索主要是针对英语等西方语系而言的。许多具有截词功能的检索系统能初步地判定词根，即指这些系统能根据一个已知的词，自动地给出这个词的单数和复数形式或基于共同词根的其他单词，并将它们作为检索词纳入检索式一并检索。

# 103 WebDewey

  • OCLC是DDC分类法的所有者，拥有全球最广泛的图书馆用户。WebDewey是DDC网络版的商业产品名称，需付费使用。
  • 它包括 DDC第23版内容以及日常更新数据，拥有体系内全功能的浏览检索、图书馆工作流集成以及外部词表的映射连接；Abridged WebDewey的分类数据来源于DDC精编第15版，主要面向中小规模馆藏的管理需要。

# 104 UDC网络版

  • UDC的网络版以机读主控文档MRF(Master Reference File)数据库驱动，采用集中统一管理模式UDC总体规模超过7万个类目，目前免费开放的UDC简本 (Summaries)包含2600多个类目，深度至多到三级类目，拥有51语言译本。UDC 网络版简本数据包含类号、类名、范围注释、应用注释、示例等信息，拥有面向普通用户访问和修订维护管理两部分。

# 105 艺术与建筑叙词表

  • AAT(Art & Architecture Thesaurus)是一部面向人文艺术领域的分面叙词表。1997年之后，不再出版纸本，只提供机读格式数据和网络检索服务。AAT提供有关人名、地点、事物的优选术语（Preferred Terms）同义词，设置了相关概念、物理属性、风格和时期、活动、材质和物件等七个分面，通过概念之间界定的语义关系构建知识库，提供丰富的领域知识与情境知识。

# 106 统一医学语言系统

  • UMLS(Unified Medical Language System) 是美国医学图书馆为医学、生物、健康专业领域设计开发的一体化主题概念系统。
  • UMLS的结构与语义复杂度早已超越传统叙词表内涵，主要包括三部分：元叙词表（Metathesaurus)、语义网络与专家词汇库。元叙词表是UMLS的核心，集成了包括MeSH、CPT（当代操作术语集）、RxNorm（美国境内临床药品规范化命名表）等在内的各种受控词表，语义网络构建了概念的语义类型与语义关系，专家词汇库主要用于自然语言处理。UMLS作为跨检索系统的词汇转换中枢，是目前语义与结构复杂度最高的专业词表。

# 107 《中分表》网络版

  • 《中分表》网络版的开发早于《中图法》网络版，于2011年正式发布，两者有一脉相承性，具有共通的数据库，逐步做到了底层数据的同步更新。目前《中分表》的分类部分来自《中图法》第五版数据，主题部分是在《汉语主题词表》基础上不断修订完善。

# 108 语义网

  • 语义网 (Semantic Web）是Web3.0的主要特征，强调计算机不仅能处理信息和显示信息，还拥有人的思考能力，即对网络信息的理解能力。语义网是对现有网络的扩展，信息拥有良好结构定义的涵义（well-defined meaning），促进 更好的人机协同工作。

# 109 世界读书日

  • 4月23日世界读书日全称“世界图书与版权日”，又称“世界图书日”。最初的创意来自于国际出版商协会。其设立目的是推动更多的人去阅读和写作，希望所有人都能尊重和感谢为人类文明做出过巨大贡献的文学、文化、科学、思想大师们，保护知识产权。

# 110 本体

  • 本体是语义网的核心，也是情报检索语言在网络环境下的最新发展。目前被普遍接受的本体（小写的ontology）定义为：共享概念模型的形式化规范说明(Gruber, 1993）。
  • 知识作为一个抽象概念以本体形式进行组织、描述、共享和使用。

# 111 本体：概念

  • 概念（Concept），也被称为类（class）。领域内所有专指概念，采用面向对象思想。
  • 概念是指拥有相同属性的一类对象的集合。对概念进行层次梳理形成本体骨架（Taxonomy）。本体的概念与之前提到的受控词表概念空间的道理相同。

# 112 本体：实例

  • 实例（Instance）是特定概念之下的具体实体。
  • 例如：若“狗”是一个概念，那么“哈士奇”就是“狗”这个概念的一个实例。
  • 根据本体建模特定需要，同一领域本体的概念与实例划分可能不同。
