# 2004-2009年阅读练题情况分析

# 一览表

# 描述性统计

报告期内共24个样本，均采用手工方式录入，忽略因计时或题型判断不清造成的误差，使用SPSS 26软件进行分析：

# 题型分布

# 总体

  • 不论是从词云图还是从统计表中都能看出细节题是最多的，占到了39.2%；
  • 推理题、作者态度题、例证题3个题型出现频率平分秋色，在10.8%-13.3%之间；
  • 中心思想题的考察是最少的，120道题中仅出现了8道。

# 第一题

第一题中出现了全部的六大题型，外加一个2005年Text1中比较特殊的“行文题”：

In the opening paragraph, the author introduces his topic by...

六大题型中，推理题和中心主旨题（考察段落主旨）仅出现1次，作者态度题仅出现2次。例证题和细节题最多，分别占29.2%和33.3%。

# 第二题

第二题中出现了五类题型，尚未见在第二题处考察中心主旨题。

例证题和作者态度题仅出现1次，推理题和细节题最多，分别占33.3%和41.7%。

# 第三题

第三题中出现了五类题型，仍尚未见在第三题处考察中心主旨题。

  • 推理题最少，出现2次；
  • 细节题最多，占41.7%；
  • 其它题型分布较为均匀，并呈现例证题和猜词题下降，作者态度题增多的趋势。

# 第四题

  • 第四题中五类题型均有考察，而绝大部分为细节题；
  • 猜词题到了第四题的位置已经降低到仅剩一篇；
  • 作者态度题到第四题处频率小小下滑一个；
  • 例证题和推理题也有出现。

# 第五题

  • 第五题中仅考察了推理题、作者态度题、中心主旨题和细节题；
  • 可以看出，作者态度、中心主旨常在第五题进行考察；
  • 最多的仍是细节题。

# 小结

  • 凭直观感受来说，在刷题联系时最最重要的是对文章细节的把握，这就要求在英语复习时要着重对词汇、长难句进行突破。根据近年命题趋势，阅读题对考生定位题干内容的要求在下降，转而对读懂长难句有了更高的要求。只有在读懂句子词汇、分析清楚长难句结构的基础上，才能够分析清楚细节题的选项。那么三省：
    • 单词背得如何了？ --> 大纲词基本背熟，第2遍单词过关即将结束。平时在真题中积累扩展词。记单词的三个渠道分别是墨墨110个左右、手写+Zotero200个左右、田静带背30个左右。
    • 语法掌握如何了？ --> 长难句基本都能划分清楚语法结构，但是像倒装句、虚拟语气等不常见的结构还是要加强学习。
    • 文章能读懂吗？ --> 基本上全文都能读懂，口译时也比较清楚，不存在什么问题。但是希望能加快阅读速度和仔细程度。
  • 例证题和猜词题多在前三题中出现；推理题多在第二、三题中出现；作者态度题多在后三题中出现；可以认为中心主旨题仅在最后一题出现，需串联全文作答。

# 做题情况

  • 做题时间均安排在下午，但时间差别较大。最早13:28，最晚20:38，平均时间为16:29。
  • 阅读完成时间存在15分钟的波动，最快能10分钟完成，最慢则需25分钟，篇均完成时间为16.7分钟。
  • 第一题均分0.75，第二题均分0.87，第三题均分0.79，第四题均分0.87，第五题均分0.79，篇均均分4.08。

# 因果关联分析

基于上述描述性统计存在的出题和答题现象，提出几项猜测并对相关性进行验证：

  • H1：做题时间越长，得分越高
  • H2：做题时间越早，得分越高

并探索六大题型出现频次对得分的影响能力。

# H1：做题时间越长，得分越高

将耗时作为自变量，而将分数作为因变量进行线性回归分析，从上表可以看出，模型公式为：

分数 = 6.187 - 0.126*耗时

对模型进行F检验时发现模型通过F检验(F=8.473，p=0.008<0.05)，也即说明耗时一定会对分数产生影响关系，最终具体分析可知： 耗时的回归系数值为-0.126(t=-2.847，p=0.009<0.01)，意味着耗时会对分数产生显著的负向影响关系。

可以认为：做题时间越长，得分越低。

# H2：做题时间越早，得分越高

做题时间早晚和得分没啥相关性，即说明做题时间并不会对分数产生影响关系，因而不能具体分析自变量对于因变量的影响关系。

# 六大题型对得分的影响

使用偏最小二乘回归（PLS回归）方法进行测度。

PLS回归用于研究多个X对于多个Y的影响关系。

上表格展示因变量（六大题型）Y与自变量（得分）X之间的回归关系表达式，包括每个因变量Y与所有自变量间的关系表达式，如下所示：

分数 = 0.172*细节题 + 0.069*推理题 + 0.472*作者态度题 + 0.207*例证题 + 0.014*中心主旨题 - 0.313*行文题 + 0.577*猜词题

但是R2和p值显示该式子的模型拟合效果一般，解释力度不强，显著性也不强。所以，只能在一定程度上作为参考。

进一步地，使用逐步回归方法（逐步stepwise），经过模型自动识别，最终余下猜词题一共1项在模型中，R方值为0.332，意味着猜词题可以解释分数的33.2%变化原因。而且模型通过F检验(F=10.957，p=0.003<0.05)，说明模型有效。以及模型公式为：

分数 = 3.648 + 0.746*猜词题

据此，得出建议：

  1. 对于当前的我，要加强对猜词题和作者态度题的练习，特别是猜词题。把这两道题做好、掌握好，可以直接逼近全对。
  2. 例证题其实已经掌握得不错了，但是要及时温习记在markji里的知识点。
