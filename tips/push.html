# 微信推送的实现

# 1 运行条件

  1. python
  2. 微信公众测试号
  3. 云服务器（使用的是宝塔面板）
  4. 已经 调试好 的代码

代码以使用以下代码获取：

git@github.com:AlexZyu/wechat-push.git

或者直接访问我的Github仓库AlexZyu/wechat-pushopen in new window下载。

# 2 步骤

# 2.1 注册微信公众测试号

  1. 利用右侧的链接注册一个微信公众测试号open in new window
  2. 扫码登录成功后，就可以生成微信公众测试号的appID和appSecret这两串数字，后面会用到
  3. 扫描测试号二维码后会生成微信号，哪个账号需要接收推送信息就需要哪个账号扫码
  4. 这里需要记住的是对应账号的微信号，也就是user id，后面需要用

# 2.2 新增测试模板

点击新增测试模板，标题自定义，内容可以使用以下模板进行修改：

{{date.DATA}} 

城市：{{city.DATA}} 
天气：{{weather.DATA}} 
最低气温: {{min_temperature.DATA}} 
最高气温: {{max_temperature.DATA}}
降雨概率：{{pop.DATA}}
今日建议：{{tips.DATA}} 
--------------
我们恋爱停留在第443天，已经{{love_day.DATA}} 天了 
距离赵宛生日还有{{birthday1.DATA}}天
--------------
每日夸夸： {{pipi.DATA}} 
--------------
今日运势： 
{{lucky.DATA}}

模板说明：
每个变量的含义如下所示，
{{date.DATA}}                  当天日期
{{city.DATA}}                  你所在的城市
{{weather.DATA}}               你所在城市的天气
{{min_temperature.DATA}}       你所在城市当天的最低气温
{{max_temperature.DATA}}       你所在城市当天的最高气温
{{pop.DATA}}                   你所在城市今天的降雨概率（需要API）
{{tips.DATA}}                  针对你所在城市的天气提出的建议（需要API）
{{love_day.DATA}}              你和对象在一起一共多少天了
{{birthday1.DATA}}             距离你对象的生日还有多少天
{{birthday2.DATA}}             距离你的生日还有多少天
{{lizhi.DATA}}                 随机出现一句励志古言（需要API）
{{note_en.DATA}}               词霸每日一句的英语
{{note_ch.DATA}}               词霸每日一句的中文
{{pipi.DATA}}                  一句彩虹屁（需要API）
可以自行添加的变量： 
{{health.DATA}}                一句健康小提示（需要API）
{{lucky.DATA}}                 今天的星座运势（需要API）

# 2.3 申请API

这里我们使用天行APIopen in new window进行使用，下面以励志古言API进行举例，会了这一个，其他的都是一样的，可以实现更多功能。

  1. 在控制台-数据管理-我申请的接口里面点击申请接口
  2. 搜索后选择励志古言，并且点击申请接口
  3. 这里我们点击右边的在线测试
  4. 这里你会得到你的key，页面不要关，这个key很重要，记下来

# 2.4 修改配置文件

提示🎗️

配置文件为config.json

  1. 双击write.exe文件，打开配置文件写入小工具
  2. 在里面填入你的所有信息，右边有相应的参考格式，不要直接复制右边的， 目前没有申请天行API的这一行先空着！！ 然后点击确定
  3. 选完了后直接点击选好了

ps. 后续配置文件config.json可以自行修改

在里面对应填入所有信息即可。其中日期要用 yyyy-mm-dd 的格式。

测试环节省略，能正常发送即可。

# 2.6 云端部署自动推送

这里使用的腾讯云服务器+宝塔面板实现的。

  1. 将源文件上传到服务器的某个看得顺眼的文件夹中
  2. 宝塔面板首页点击计划任务
  3. 添加计划任务，任务类型为“Shell脚本”，设置好运行周期和时间
  4. 添加以下两行脚本内容：
cd /www/wwwroot/push # 源文件根目录
python3 main.py
  1. 点击添加任务！

搞定 🎉🎉🎉
